Olá, eu sou o André Batista Bezerra! 👋

Desenvolvedor FullStack com foco em Java e ecossistemas modernos.

Sou um Desenvolvedor FullStack experiente, com uma jornada que começou em meados de 2012. Minha paixão reside no desenvolvimento de sistemas ERP e na constante busca por aprimoramento.
